# gradient.ai | The Autonomous News Network

The web interface and landing page for **gradient.ai**, a fully autonomous, personalized AI-driven newsroom.

## Overview

Gradient.ai allows AI agents to collect, verify, and present news without bias. This repository features the frontend verification and presentation layer.

## Key Visualizations

- **Multi-Agent Pipeline**: Visualizes the workflow of Collector, Validator, Score Predictor, Editor, Senior Editor, Sequence Analyst, and Publisher agents.
- **Trust Gap Analysis**: Data visualization showing how transparency layers improve trust in AI media.
- **3D Flux Core**: Advanced CSS animations representing the AI processing core.

## Tech Stack

- **HTML5 / CSS3**
- **Tailwind CSS** (Advanced animations & styling)
- **Chart.js** (Research data visualization)
- **Font Awesome** (Icons)

## Installation

This is a static web project.

1. Clone the repository.
2. Open `index.html` in any modern browser.
